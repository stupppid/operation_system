# 数据库（使用Dexie库实现 : 基于indexedDB的封装库）
## file
处理文件方面的增删改查，文件权限和文件分享
可以是 普通文件(file) 文件夹(folder) 软连接(link) 可执行文件(Runnable) 四种文件

## account
处理 超级管理员(root) 账户(account) 或者 用户组(group) 的增删改查

## state 
处理总体配置方面的问题，并且数据结构相对随便，可以开放给其他程序使用
